# API Response Debugger Example

This example demonstrates how to use the API Response Debugger package in a Flutter application.

## Getting Started

1. Run the example:
```bash
flutter run
```

2. Use the buttons to:
- Make an API call to a test endpoint
- Open the debug dashboard to inspect the API responses

The example shows:
- How to initialize the API Response Tracker
- How to track API responses
- How to integrate the debug dashboard
- How to handle success and error cases
